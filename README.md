# Secure Reliable File Transfer

This repository is a project showcase for a collaborative course project focused on secure and reliable file transfer over UDP using raw sockets.

## Project Overview

The project involved designing and testing a custom file transfer protocol that added reliability and security features above UDP. The system included packet construction, retransmission logic, checksums, sequence numbers, acknowledgements, and a secure layer for authentication and encrypted transfer. The project was developed and tested in a Docker-based environment.
